## アプリケーション名
見積アプリ

## アプリケーション概要
計画を実行する上で必要な費用とその内訳を、計画毎に作ったルームで投稿し、ユーザーと共有することができる。

## 利用方法
### 見積投稿
1.トップページよりユーザー新規登録を行う
2.見積を作成するボタンから、見積ルーム名を入力し、チャットメンバーを選択する
3.見積ルームが作成できたら、見積の内容（内容・金額・メモ）を入力し投稿する
